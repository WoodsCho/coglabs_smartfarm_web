💡 핵심 원칙
• 모든 IoT 디바이스는 AWS와 직접 통신하지 않습니다. 반드시 MiniPC 엣지를 거쳐 MQTT로 송수신합니다.
• 실시간 제어는 Browser → (Tailscale) → MiniPC Control API → MQTT → Device 경로로 저지연 보장.
• 이력 데이터는 Device → MQTT → Edge Bridge → AWS IoT Core → Lambda → DynamoDB 경로로 적재되고, Browser는 REST(API Gateway)로 조회합니다.
• 네트워크 단절 시 MiniPC가 로컬 브로커 + 로컬 DB로 자체 운영을 지속하고, 연결 복구 시 AWS로 백필합니다.
